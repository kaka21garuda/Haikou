## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm
- Docker
- PostgreSQL
- Redis

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/haikou.git
cd haikou
```

2. Install dependencies:
```bash
pnpm install
```

3. Set up environment variables:
```bash
cp .env.example .env
```

4. Start the development environment:
```bash
pnpm dev
```
